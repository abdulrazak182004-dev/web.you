Web Profile Saya

Website ini dikembangkan sebagai media untuk memperkenalkan diri sekaligus menunjukkan kemampuan dasar saya dalam Mata Kuliah Interaksi Manusia dan Komputer (Praktik), khususnya pada pembuatan tampilan web menggunakan HTML dan CSS. Melalui proyek ini, saya mempraktikkan pemahaman mengenai struktur halaman web, penataan gaya, serta perancangan antarmuka yang bersih dan mudah dipahami. Selain itu, proyek ini juga merupakan bagian dari proses pembelajaran saya sebagai Mahasiswa Sistem Informasi semester 3 dalam menerapkan konsep dasar pengembangan web secara mandiri, terarah, dan terstruktur.

Website ini terdiri dari tiga halaman: Beranda, Tentang, dan Kontak.
